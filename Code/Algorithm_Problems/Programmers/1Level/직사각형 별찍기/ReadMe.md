```
    문제 설명
    이 문제에는 표준 입력으로 두 개의 정수 n과 m이 주어집니다.
    별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

    제한 조건
    n과 m은 각각 1000 이하인 자연수입니다.
    예시
    입력

    5 3
    출력

    *****
    *****
    *****
```

- 풀이 코드

```cpp
    #include <iostream>

    using namespace std;

    int main(void) {
        int a;
        int b;
        cin >> a >> b;
        for(int i = 0; i < b; i++)
        {
            for(int n = 0; n < a; n++)
            {
                cout << '*';    
            }
            cout << endl;
        }
        return 0;
    }
```

- 주어진 정수 두개를 이용해서 별 직사각형을 출력하는 코드이다.
- 이러한 별 찍기는 처음 코딩을 배웠을 때부터 지금까지 계속 해왔기에 매우 간단히 풀 수 있었다.
